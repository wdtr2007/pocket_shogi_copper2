This is a javascript game that currently runs on 1 computer with two people using that 1 computer.

It is a variant of shogi, and has a 2nd drop zone called a pocket.

The project is new, and I hope to make a network/web version of this within the next 6 months.

If you like chess, I encourage you to play a few games of this and learn the basic moves.
I think it is one of the better chess variants out there.
